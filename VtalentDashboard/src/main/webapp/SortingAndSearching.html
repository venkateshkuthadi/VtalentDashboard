Problem 1: Implement Bubble Sort

Description

Write a Java function to implement the Bubble Sort algorithm to sort an integer array in ascending order.

Function Signature

public static void bubbleSort(int[] arr)

Requirements:

  • Sort the array in-place using the Bubble Sort algorithm
  • Optimize the algorithm to stop early if the array is already sorted
  • Handle edge cases (empty array, null array)

Example:

Input: [64, 34, 25, 12, 22, 11, 90]
Output: [11, 12, 22, 25, 34, 64, 90]
