Problem 1: Right-Angled Triangle Pattern

Description

Write a Java function to print a right-angled triangle pattern using asterisks (*).

Function Signature

public static void printRightTriangle(int n)

Requirements:

  • Print a right-angled triangle with n rows
  • Each row should have the appropriate number of asterisks

Example (n=5):

*
**
***
****
*****
