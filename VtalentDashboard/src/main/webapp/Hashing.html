Problem 1: Two Sum

Description

Write a Java function to find indices of two numbers in an array that add up to a specific target using hashing.

Function Signature

public static int[] twoSum(int[] nums, int target)

Requirements:

  • Use a HashMap to store values and their indices
  • Return indices of the two numbers that add up to the target
  • Assume exactly one solution and you may not use the same element twice

Example:

Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1] (because 2 + 7 = 9)

How to use the compiler:

  1. Write your solution in the code editor on the right
  2. Click the "Run" button in the Programiz compiler to execute your code
  3. Check the output in the compiler's output panel
