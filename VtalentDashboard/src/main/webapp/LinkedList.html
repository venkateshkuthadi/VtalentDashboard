Problem 1: Reverse a Linked List

Description

Write a Java function to reverse a singly linked list.

Node Class Definition

class Node {
    int data;
    Node next;
    Node(int data) { this.data = data; }
}

Function Signature

public static Node reverseLinkedList(Node head)

Requirements:

  • Reverse the linked list in-place
  • Return the new head of the reversed list
  • Your solution should have O(n) time complexity

Example:

Input: 1 -> 2 -> 3 -> 4 -> 5 -> NULL
Output: 5 -> 4 -> 3 -> 2 -> 1 -> NULL

How to use the compiler:

  1. Write your solution in the code editor on the right
  2. Click the "Run" button in the Programiz compiler to execute your code
  3. Check the output in the compiler's output panel
