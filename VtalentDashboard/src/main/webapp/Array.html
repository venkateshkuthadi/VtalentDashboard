Problem 1: Find Maximum Value

Description

Write a Java function to find the maximum value in an integer array.

Function Signature

public static int findMax(int[] arr)

Requirements:

  • Return the maximum value in the array
  • Handle edge cases (empty array, null array)

Example:

Input: [3, 5, 7, 2, 8, -1, 4, 10, 12]
Output: 12
