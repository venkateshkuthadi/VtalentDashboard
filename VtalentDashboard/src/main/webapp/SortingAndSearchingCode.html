Problem 1: Binary Search

Description

Write a Java function to implement binary search on a sorted integer array.

Function Signature

public static int binarySearch(int[] arr, int target)

Requirements:

  • Return the index of the target element if found
  • Return -1 if the target is not in the array
  • Assume the input array is sorted in ascending order

Example:

Input: [1, 3, 5, 7, 9, 11, 13, 15], target = 9
Output: 4

How to use the compiler:

  1. Write your solution in the code editor on the right
  2. Click the "Run" button in the Programiz compiler to execute your code
  3. Check the output in the compiler's output panel
